# Image Resizer using OpenCV

A simple Python script to resize an image using OpenCV.

## Features
- Resize image by percentage
- Supports JPG, PNG, and other common formats
- Saves resized image as a new file

## Usage
1. Place the image in the project folder
2. Update the image name in the code
3. Run:
   python main.py

## Output
A resized image is saved in the same directory.
